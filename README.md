## Features

- Draw using index finger movements  
- Color selection (multiple colors)  
- Eraser and Erase-All functionality  
- Adjustable pen thickness  
- Dual screen (webcam view + whiteboard view)  
- Gesture-based selection using fingertips  
- Real-time hand tracking using MediaPipe
